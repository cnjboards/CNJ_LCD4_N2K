# CNJ_LCD4_N2K

This is very much a work in progress. This code runs on my custom designed board.
The display is about ~400nits brightness which is good for indoor use.
This works well indoors but not in a sunlight environment. 
I am working on a ~800-1000 nits version for outdoor use. 
I have a shop over on Tindie if your interested in buying hardware.
Otherwise, cnjboards "at" gmail "dot" com works fine.

Key Features:

- 12-24Vdc Powered
- NMEA2000 Interface
- 400 Nits 480 x 480 RGB display, with pwm brightness
- I2C Interface
- Onewire Interface
- TCA9554 expansion IO
- LVGL graphics, touch screen interface
- 3D printed case design
